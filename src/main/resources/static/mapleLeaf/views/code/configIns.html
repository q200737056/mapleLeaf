配置文件说明 {*必需：必需配置该标签（属性），内容可以为空；非必需：可以不配置该标签（属性）}

configure根标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
global 全局配置 否 没有
db 连接数据库的配置，详情请看db标签说明 是 没有
module 模块设置，可以配置多个module，详情请看module标签说明 是 name 属性name为模块名，默认""，非必需

global标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
property 全局属性配置，可配置多个 否 name,value 配置name-value，现有属性：baseDir(路径,默认项目所在目录),basePackage(基础包名，默认mapleLeaf.code) ,baseColPrefix(需要去掉的全局字段名前缀，默认""),baseTabPrefix(需要去掉的全局表名前缀，默认"") ,columnCamel(表字段下划线转驼峰命名，默认true),wrapperClass(java基本类型是否为包装类，默认true) ,persistence(持久层框架，默认mybatis)
common 公共类配置，形式：“包名=模板”多个逗号分隔，模板不需要后缀。如：util=Page,commom.Dao=BaseDao 否 没有

db标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
dbType 数据库 是 没有 可选：mysql，oracle，postgresql，sqlserver，informix
dbName 数据库名 是 没有
user 数据库用户名 是 没有
pwd 数据库密码 是 没有
driver 数据库驱动 是 没有
url 数据库url 是 没有

module标签（可配置多个）

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
table 配置数据库表，一个模块下可以配置多个table 是 tabName,entName,exclude tabName属性：表名，必需； entName属性：自定义表对应的类名，非必需； exclude属性：排除指定模板代码生成（entity,dao,service,controller,custom选择多个时，逗号分隔），非必需；
package 包名配置 是 没有
customArea 自定义区域。详情请查看customArea标签 否 没有

package标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
property 包名属性配置，可配置多个（如果为空或没配置，则该包对应类不会生成代码 ） 否 name,value 配置name-value，entityPkg(实体类包名),daoPkg(dao包名，实现类默认该包的impl下) ,servicePkg(业务层类包名,实现类默认该包的impl下),controllerPkg(控制层类包名) ,mapperPkg(mybatis的mapperXML包名)

customArea标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
codeFile 自定义配置，可配置多个;标签内容为模板，多个逗号分隔，必需 否 customPkg,suffix customPkg属性：配置包名，必需；suffix属性：配置生成代码文件的后缀，非必需，默认ctm
param 自定义参数，可配置多个 否 key,value key属性：参数名，必需；value属性：参数值，必需；模板中可通过${param['key']!}获取参数值

table标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
ref 关联表配置（单向）可以配置多个ref，如果需要双向关联，需两个table相互配置ref。 标签内容为 主表字段=关联表字段，可以多个，如：id=id,name=user_name； 特殊情况：多对多时，标签内容为 主表字段=中间表字段。 否 tabName，type,midTabName,midRefCol,forKey tabName属性：关联表的表名，而且该表名必须配置table，必需； type属性：关联关系（OneToOne，ManyToMany，ManyToOne，OneToMany），必需； midTabName属性：中间表名，不用配置table，多对多时必需； midRefCol属性：中间表字段=关联表字段，多对多时必需； forKey属性：主表中的关联字段（外键），多对一时必需，一对一时，判断关联字段（外键）是否在主表中，不在就不用配置。
columnGroup 表字段集合 否 exclude，searchPos，listPos，inputPos exclude属性：需要排除的表字段（一般不要去排除），非必需； searchPos属性：存放于searchColumns，页面的查询条件，多个字段逗号分隔，按输入顺序排序，非必需； listPos属性：存放于listColumns，页面的查询结果列表，多个字段逗号分隔，按输入顺序排序，非必需； inputPos属性：存放于inpurColumns，页面的表单输入项，多个字段逗号分隔，按输入顺序排序，非必需；

columnGroup标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
column 表字段标签，可自定义字段属性；配置的字段名必须未被过滤掉，而且表中存在； 标签内容为字段值设定，比如 flag字段表示状态。可自定义字段值 如：0=可用,1=不可用；对应COMMENT约定格式(字段文本;表单类型;val1:text1,val2:text2;)中的第3个； 默认会从COMMENT中取（如果有）。 否 colName，labelName，tagType，propName colName属性：字段名，必需； labelName属性：自定义字段文本，可用来在页面中显示文本，非必需， 默认会从COMMENT中取； tagType属性：自定义表单标签类型,可选(text,select,date,checkbox,radio,hidden,textarea,file) ，默认 text，非必需，默认会从COMMENT约定格式(字段文本;表单类型;val1:text1,val2:text2;)中取； propName属性：自定义java类的属性名。
