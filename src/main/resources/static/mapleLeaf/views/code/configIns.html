配置文件说明 {*必需：必需配置该标签（属性），内容可以为空；非必需：可以不配置该标签（属性）}

configure根标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
basedir 生成代码文件的路径。如：d:\\code 是 没有
basePackage 生成代码的基础包名。如：com.test 是 没有
isDeleteTablePrefix 是否删除表名前缀。（true或false)默认false 否 没有
baseTabPrefix 全局表名前缀，可以多个英文逗号分隔，如：sys_,log_。默认"" 否 没有
columnIsCamel 是否表字段中下划线转成驼峰命名。（true或false)默认false 否 没有
persistance 持久层框架。（mybatis,hibernate,other) 是 没有
common 公共类，形式：“包名=模板”多个英文逗号分隔，模板不需要后缀。主要是一些工具类，父类等,直接输出生成java文件，一般不需要。如：util=Page,commom.Dao=BaseDao 否 没有
db 连接数据库的配置，含有子标签，详情看db标签说明 是 没有
module 模块设置，可以配置多个module，详情看module标签说明 是 没有

db标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
dbType 数据库。（mysql,oracle,sqlserver,informix,postgresql） 是 没有
dbName 数据库名 是 没有
user 数据库登陆用户 是 没有
pwd 数据库登陆密码 是 没有
driver 数据库驱动 是 没有
url 数据库url 是 没有

module标签（可配置多个）

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
name 模块名。默认 "" 否 没有
table 配置数据库表，一个模块下可以配置多个table 是 entityName,name, prefix,exclude entityName属性：配置自定义实体类名，非必需； name属性：表名，必需；prefix属性：表名前缀，非必需； exclude属性：排除指定模板代码生成（entity,dao,service,controller,view,custom选择多个时，英文逗号分隔），非必需；
entityPackage 自定义实体类包名。如：ai.entity。如未配置或标签内容为空字符，则该模块下不会去生成实体类，如下一样 否 没有
daoPackage 自定义dao接口包名,实现类默认在该包的impl下。如：ai.dao。 否 没有
servicePackage 自定义service接口包名,实现类默认在该包的impl下。如：ai.service。 否 没有
controllerPackage 自定义controller类包名。如：ai.controller。 否 没有
mapperPackage 自定义mapperXML包名。如：mapper。 否 没有
viewPackage 自定义视图层包名。如：ai.jsp。 否 tpl，suffix tpl属性：配置模板，只需模板名，不需要后缀，多个可以用英文逗号分隔，如view,list。必需属性; suffix属性：配置生成文件类型，即文件后缀，如jsp。非必需属性，默认‘jsp’
customPackage 自定义(自定义生成文件)包名。如：ai.custom。 否 tpl，suffix tpl属性：配置模板，只需模板名，不需要后缀，多个可以用英文逗号分隔，如indexJs。必需属性; suffix属性：配置生成文件类型，即文件后缀，如js。非必需属性，默认‘ctm’

table标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
ref 关联表配置（单向）可以配置多个ref，如果需要双向关联，需两个table相互配置ref。 标签内容为两table关联的字段，table标签的字段在前，可以多个，如：id=id,name=user_name 否 name，type name属性：关联表的表名，而且该表名已配置table，必需； type属性：关联关系（OneToOne，ManyToMany，ManyToOne，OneToMany），必需
columnGroup 表字段过滤，默认加载全部表字段 否 include，exclude include属性：需要包含的表字段，非必需； exclude属性：需要排除的表字段，非必需；如果两者都存在，只会生效exclude排除属性

columnGroup标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
column 表字段标签，可自定义字段属性；配置的字段名必须未被过滤掉，而且表中存在； 标签内容为字段值设定，比如 flag字段表示状态，表中存的是标识。 如：0=可用,1=不可用；默认是从COMMENT中取 COMMENT约定形式(字段文本;val1:text1,val2:text2;) 否 colName，labelName，tagType colName属性：字段名，必需； labelName属性：自定义字段文本，可用来在页面中显示文本，非必需， 默认是从COMMENT中取 COMMENT约定形式(字段文本;val1:text1,val2:text2;) tagType属性：自定义表单标签类型，页面中可使用，可以自己定义一套， 比如 select表示下拉框，checkbox表示多选，radio表示单选等，非必需
