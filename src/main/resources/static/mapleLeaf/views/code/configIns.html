配置文件说明 {*必需：必需配置该标签（属性），内容可以为空；非必需：可以不配置该标签（属性）}

configure根标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
global 全局配置 否 没有
db 连接数据库的配置，详情请看db标签说明 是 没有
module 模块设置，可以配置多个module，详情请看module标签说明 是 name 属性name为模块名，默认""，非必需

global标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
property 全局属性配置，可配置多个 否 name,value 配置name-value，现有属性：baseDir(路径,默认项目所在目录),basePackage(基础包名，默认mapleLeaf.code) ,deleteTabPrefix(是否删除表名前缀，默认false),baseTabPrefix(全局表名前缀，默认"") ,columnCamel(表字段下划线转驼峰命名，默认false),persistance(持久层框架，默认mybatis)
common 公共类配置，形式：“包名=模板”多个逗号分隔，模板不需要后缀。如：util=Page,commom.Dao=BaseDao 否 没有

db标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
property 数据库配置，可以配置多个 是 name，value 配置name-value： dbType（数据库），dbName（数据库名），user（用户名），pwd（密码）， driver（数据库驱动），url（数据库url）；这些项都必需

module标签（可配置多个）

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
table 配置数据库表，一个模块下可以配置多个table 是 entityName,name, prefix,exclude entityName属性：配置自定义实体类名，非必需； name属性：表名，必需；prefix属性：表名前缀，非必需； exclude属性：排除指定模板代码生成（entity,dao,service,controller,view,custom选择多个时，逗号分隔），非必需；
entityPackage 自定义实体类包名。如：ai.entity。如未配置或标签内容为空字符，则该模块下不会去生成实体类，如下一样 否 没有
daoPackage 自定义dao接口包名,实现类默认在该包的impl下。如：ai.dao。 否 没有
servicePackage 自定义service接口包名,实现类默认在该包的impl下。如：ai.service。 否 没有
controllerPackage 自定义controller类包名。如：ai.controller。 否 没有
mapperPackage 自定义mapperXML包名。如：mapper。 否 没有
viewPackage 自定义视图层包名。如：ai.jsp。 否 tpl，suffix tpl属性：配置模板，只需模板名，不需要后缀，多个可以用逗号分隔，如view,list。必需属性; suffix属性：配置生成文件类型，即文件后缀，如jsp。非必需属性，默认‘jsp’
customPackage 自定义(自定义生成文件)包名。如：ai.custom。 否 tpl，suffix tpl属性：配置模板，只需模板名，不需要后缀，多个可以用逗号分隔，如indexJs。必需属性; suffix属性：配置生成文件类型，即文件后缀，如js。非必需属性，默认‘ctm’

table标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
ref 关联表配置（单向）可以配置多个ref，如果需要双向关联，需两个table相互配置ref。 标签内容为两table关联的字段，table标签的字段在前，可以多个，如：id=id,name=user_name 否 name，type name属性：关联表的表名，而且该表名已配置table，必需； type属性：关联关系（OneToOne，ManyToMany，ManyToOne，OneToMany），必需
columnGroup 表字段集合 否 exclude exclude属性：需要排除的表字段，非必需；

columnGroup标签

子标签 子标签说明 是否必需 子标签属性 子标签属性说明
column 表字段标签，可自定义字段属性；配置的字段名必须未被过滤掉，而且表中存在； 标签内容为字段值设定，比如 flag字段表示状态。可自定义字段值 如：0=可用,1=不可用；对应COMMENT约定格式的第3个； 默认会从COMMENT中取（如果有） COMMENT约定格式(字段文本;表单类型;val1:text1,val2:text2;) 否 colName，labelName，tagType colName属性：字段名，必需； labelName属性：自定义字段文本，可用来在页面中显示文本，非必需， 默认会从COMMENT中取 tagType属性：自定义表单标签类型，页面中可使用，可以自己定义一套， 比如 select表示下拉框，checkbox表示多选，radio表示单选等，非必需， 默认也会从COMMENT中取
