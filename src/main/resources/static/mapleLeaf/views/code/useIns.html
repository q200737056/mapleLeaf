注意事项

1.本项目运行不依赖数据库，推荐使用maven打成jar包部署运行，在jar所在目录下加入项目中的配置文件config.xml跟模板文件夹template。
2.数据库建表时，最好写上COMMENT注释；并且按约定的格式注释（格式：字段文本;表单类型;val1:text1,val2:text2;）比如有个flag字段, COMMENT 状态;select;0:可用,1:不可用
3.推荐每张表至少建立一个主键（不推荐组合主键）或唯一索引（复合索引），建议索引名取的好一点。
4.模板文件与配置文件请保持一致，否则出错，生成代码失败。
5.同模块中表不要重复配置，不然生成的代码数据会混乱，不正确。不同模块请配置不同包名。

模板使用说明

1.本项目模板是使用freemarker编写的，只要会点freemarker，编写模板也很容易。
2.模板可用数据说明（如下表，可查看参考项目中的默认模板）。

属性 类型 说明 示例
基础属性
basePackage String 基础包名 ${basePackage}输出基础包名，如下一样
entityPackage String 实体类包名
servicePackage String service接口包名
serviceImplPackage String service实现类包名，默认impl
daoPackage String dao接口包名
daoImplPackage String dao实现类包名，默认impl
controllerPackage String controller类包名
mapperPackage String mapperXML包名
persistence String 持久层框架 可以用来判断，mybatis or hibernate
Table属性
tabName String 表名 ${tabName}就可以输出表名，如下一样
lowEntName String 首字母小写的实体类名，下划线表名会自动转成驼峰命名
entName String 对应表的实体类，下划线表名会自动转成驼峰命名
remark String 表comment
impClasses List<String> 需要导入字段类型的全类名集合 如：java.util.Date
columns List<Column> 表字段，属性等（详见Column属性） 利用<#list columns as col>freemarker标签， 可循环取出Column属性
refTables List<RefTable> 关联表的信息（详见RefTable属性） 利用list标签, 循环取出RefTable属性
uniIdxMap Map<String,List<Column>> 主键，唯一索引map集合；key：主键名或索引名，value:Column集合 利用list标签, 循环取出key和主键或唯一索引的Column集合
RefTable属性
tabName String 表名（同Table）
lowEntName String 首字母小写的实体类名，下划线表名会自动转成驼峰命名（同Table）
entName String 对应表的实体类，下划线表名会自动转成驼峰命名（同Table）
columns List<Column> 表字段，属性等（详见Column属性）（同Table）
refType String 关联属性(OneToMany,OneToOne,ManyToOne,ManyToMany)
refColMap Map<String,String> 关联字段，key主表字段，value从表字段
refPropMap Map<String,String> 关联字段对应的属性，key主表实体类属性，value从表实体类属性
Column属性
colName String 表字段名
colType String 如果是mybatis则为jdbcType,hibernate则为hibernate的映射类型
remark String 表字段comment,可用来注释
propName String 字段对应的实体类属性名
propType String 属性类型，即字段类型对应的java类型
upperPropName String 属性名首字母大写，主要用来get..,set..
nullable Boolean 表字段是否为空
pk Boolean 表字段是否为主键
length String 表字段文本类型的长度
defaultValue Object 表字段默认值
labelName String 表字段文本，对应COMMENT格式（字段文本;表单类型;val1:text1,val2:text2;）第1个 ，可在配置文件column标签自定义字段文本 可用在页面模板上
tagType String 表单标签类型，对应COMMENT格式（字段文本;表单类型;val1:text1,val2:text2;）第2个 ，可在配置文件column标签自定义表单类型
colValueMap Map<String,String> 表字段值，对应COMMENT格式（字段文本;表单类型;val1:text1,val2:text2;）第3个， 可在配置文件中column标签内容中自定义的字段值
